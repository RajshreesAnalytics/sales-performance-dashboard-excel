# Sales Performance Dashboard (Excel)

## Overview
This project presents an interactive sales performance dashboard built using Microsoft Excel. It analyzes revenue trends, product performance, sales representative contribution, and customer segments.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Metrics

## KPIs
- Total Revenue
- Total Units Sold
- Average Revenue per Unit
- Revenue Growth %
- Top Product by Revenue
- Revenue Concentration Ratio

## Key Insights
- Revenue increased by 82% from January to March.
- Product P001 contributes 41.6% of total revenue.
- Sales Rep R001 generates 42.8% of total revenue.
- Corporate customers contribute 58% of revenue.
